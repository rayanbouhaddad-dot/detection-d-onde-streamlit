# 🌊 WavePicker

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://share.streamlit.io/VOTRE_NOM_ICI/wavepicker/main/streamlit_picker.py)

Interface web pour le picking manuel des ondes P et S sur des signaux ultrasoniques.

## 🎯 Comment ça marche ?

1. Entrez le chemin du dossier contenant vos fichiers `CH1.txt`
2. Cliquez sur "Charger"
3. Sur le graphique : **Clic gauche** → P (rouge) | **Clic droit** → S (bleu)
4. Naviguez entre les fichiers avec "Suivant" / "Précédent"
5. Sauvegardez les résultats en CSV
